# sprints/

Sprint-scoped contracts and evaluation rounds from the pre-release
sprint cycle (Sprint 7 variants: `7a`, `7b`, `7b-baseline`, `7c`).

This directory is **entirely archived** — every entry lives under
`archive/`, kept as historical record of how those sprints were scoped
and evaluated. Not part of the build, not linked from the live site.
Later releases are tracked by semver (`.checkpoints/`, `/changelog`,
`skills/CHANGELOG.md`) rather than sprint number.
