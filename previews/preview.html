opchain

skills that ship.

a skillchain and checkpoint protocol designed to get you from concept to production to ops

If you've ever tried to build something real with Claude and watched it lose the plot between chats — forgetting your spec, redesigning what it already designed, repeating questions it asked an hour ago — opchain is what you've been looking for.

A set of Claude Code skills plus a checkpoint protocol. Each skill specializes in one piece of the puzzle (discovery, spec, design, build, audit, ship, monitor, migrate, scale). At the beginning of the next session, Claude reads them back to pick up right where you left off. A central orchestrator routes between them so you never have to think about which skill to call.

Install them, trigger by name, keep going. No API keys. No SaaS. No vendor lock-in — every skill is a single Markdown file.

app-architect

Unified app development from idea to launch. Runs the Generator/Evaluator QA loop. Auto-invokes stack-forge and ux-engineer at the relevant phases.

app-architect is the orchestrator skill at the front of the pipeline. It owns the conversation from a vague prompt ("I want to build X") through to a deployable app, delegating to specialists at each checkpoint.

It pairs with git-ops and deploy-ops on the back end of the loop.
