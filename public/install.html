install

Three flows.

download bundle

Every skill is a single SKILL.md file. Install all ten or just the ones you want. No API keys, no package manager, no build step.

Claude Code CLI

Drop the files into your project's .claude/skills/ directory. Claude Code picks them up on the next session.

curl -L https://opchain.dev/opchain-skills.zip -o opchain-skills.zip
unzip opchain-skills.zip -d .claude/skills/
claude
> /discover

Claude.ai / Claude Desktop

  1. Download individual SKILL.md files from the Skill Library, or the full bundle ZIP.
  2. Open Claude → Settings → Customize → Skills.
  3. Click Upload skill and pick the .md file.
  4. Start a new conversation and trigger the skill by name (e.g. "I have an app idea" → auto-invokes app-architect) or by slash command (/discover).

Team (check into git)

Commit the skills directory so every teammate uses the same prompt pipeline.

git add .claude/skills/
git commit -m "chore: add opchain skills"
git push
