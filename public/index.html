opchain

10 interconnected skills · 1 protocol skills that ship — concept to production

opchain is a set of Claude Code skills that form a software development pipeline. Each skill owns one phase — discovery, spec, design, build, audit, ship, scale. Skills share state through .checkpoints/ JSON files so work persists across conversations and Claude picks up where you left off.

Install a skill, trigger it by name, and pick up where you left off. No API keys, no SaaS, no vendor lock-in.
