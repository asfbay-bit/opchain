v1.4.2 Complete bundles, honest checkpoints 2026-06-05

Emergency patch: the "download all" bundle was silently shipping only bare SKILL.md files, dropping the references/, scripts/, and examples/ that make a skill actually run. Alongside the fix: oc- canonical aliases, a unified checkpoint schema, and automated drift detection.

  • Complete skill bundles. opchain-skills.zip now bundles full skill trees.
  • oc- canonical names. Every skill gets an oc-<name> alias.
  • Checkpoint schema unified. Validator, spec, and README now agree on one wire format.
  • Drift detection. checkpoint:doctor cross-checks against git log.
