New “AI” phase — Phase 3 of 6
new band · inserted between Plan and Build

v1.5 (“Build the AI app”) shipped four AI-native skills that own model routing, retrieval, agents, and prompts. They declare phases: [build, ai-native] in their own frontmatter, so they earn a dedicated phase rather than being crammed into Build. They are auto-invoked branches off oc-app-architect’s “AI app?” decision in /oc-discover — the same branch pattern Build skills already use — so they return checkpoints to oc-app-architect and do not take primary-spine ordinals.
