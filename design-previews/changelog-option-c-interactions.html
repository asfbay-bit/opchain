v1.4.2 Complete bundles, honest checkpoints Jun 05, 2026

Emergency patch: the "download all" skill bundle was shipping only bare SKILL.md files, dropping the directories that make a skill run. Fixed, plus oc- aliases, unified checkpoint schema, drift detection.

patch oc- aliases 18 skills checkpoint

What's new

  • "Download all" ships complete skills again. opchain-skills.zip now bundles full skill trees, matching per-skill downloads.
  • oc- canonical aliases. Every skill gets an unambiguous oc-<name> form. Bare names still resolve.
  • Orchestrator stops misreporting itself. Persistence docs rewritten to match git-tracked .checkpoints/ reality.
  • One checkpoint schema, enforced. Validator, spec, and README now agree on a single wire format. next_actions required while in-progress.
  • Drift detection. checkpoint:doctor cross-checks against git log + live /api/health.

New CLI commands

  • checkpoint:doctor — cross-checks checkpoints against git + health endpoint
  • checkpoint:next — "what should I do next" without a registry
  • checkpoint:status --brief / --since=<date> — terse resume + momentum digest
  • checkpoint:init — deterministic cold-start scaffold

pack registry 32 packs mobile dispatch

Pack registry

One pack.yml per language, framework, mobile platform, or deploy target. scripts/gen-stack-packs.mjs validates every pack on every build — bidirectional language ↔ frameworks invariants, ref-doc budgets, coverage-flag emission.

  • 6 deploy targets: Cloudflare, Fly.io, Render, Railway, Netlify, Heroku, AWS Amplify, App Store, Play Store

v1.3 Runtime PM, real platforms, automated releases May 11, 2026

PM-MCP loop becomes executable end-to-end. release-ops ships as the 18th skill. Platform menu expands to Django, Rails, Go, Rust.

release-ops PM-MCP 18 skills

What's new

  • PM-MCP loop is real. Concrete tool-name registry, retry/backoff, idempotency markers, deferred-action queue.
  • release-ops (18th skill). /release plan|draft|bump|announce|ship. Reads checkpoints to propose next semver, drafts changelog entry from sprint outputs.
  • Platform menu grew. Django + Render, Rails + Heroku, Go + Fly.io, Rust + Shuttle.rs — first-class in all three relevant skills.
  • Build-time PM-MCP validator. Blocks builds on placeholder drift or unknown tool names.
