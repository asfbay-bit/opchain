scenario 03 · audit gate before deploy

Catch a SQL injection at the production gate.

A solo founder pushes a Stripe checkout patch on Friday afternoon. opchain's audit gate runs both auditors before the deploy fires — finds an injection, hands the user a one-line patch, then ships.
